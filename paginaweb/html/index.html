Introduccion

Este sitio web fue creado como parte de un proyecto de desarrollo web. El objetivo es demostrar el uso correcto de etiquetas HTML semanticas, tablas, listas, enlaces, CSS externo y JavaScript para interaccion con el DOM. La tematica elegida es el popular juego Minecraft.

Normas del Blackjack

  1. El objetivo es conseguir una mano con un valor lo mas cercano posible a 21, sin pasarse.
  2. Las cartas numericas valen su valor nominal, las figuras valen 10 y el As puede valer 1 u 11.
  3. Se reparten dos cartas al jugador y al crupier.
  4. El jugador puede pedir cartas ("hit") o plantarse ("stand").
  5. Si la mano supera 21 puntos, el jugador pierde automaticamente ("bust").
  6. Despues de que el jugador termina, el crupier juega siguiendo reglas fijas (normalmente se planta con 17 o mas).
  7. Gana quien tenga la mano mas cercana a 21 sin pasarse.
